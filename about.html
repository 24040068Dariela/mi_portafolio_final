Objetivo del Portafolio

Este portafolio ha sido desarrollado como parte de la materia de Frameworks para demostrar la aplicación práctica de conceptos de desarrollo web. El objetivo es integrar los conocimientos adquiridos durante el curso en un proyecto funcional que muestre el dominio de:

HTML5
Estructura semántica
CSS3
Estilos y animaciones
JavaScript
Interactividad
Tailwind
Framework CSS
  • Diseño y desarrollo de interfaces de usuario (Frontend)
  • Implementación de navegación y componentes reutilizables
  • Diseño responsivo para múltiples dispositivos
  • Gestión de rutas y enlaces dinámicos con configuración centralizada
  • Desarrollo en Visual Studio Code con extensiones especializadas
